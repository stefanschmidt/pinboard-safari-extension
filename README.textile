h2. Description

PinSafari is a Safari extension for saving bookmarks to "Pinboard":http://pinboard.in/ originally written by "Jake Bell":https://github.com/theunraveler/.

h2. Requirements

Safari 5.0 or greater

h2. Usage

1. Install the current "release":https://github.com/stefanschmidt/Pinboard.in-Safari-Extension/downloads
2. Right-click the Safari toolbar and select "Customize Toolbar..." to add the PinSafari icon to the toolbar
